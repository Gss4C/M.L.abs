In questa cartella ci sono notebook utilizzati su Colab, che
mette a disposizione una grande quantità di risorse gratuite
come GPU e CPU molto potenti così da poter sperimentare le
funzionalità dei transformers.
Questi file python sono l'unione brute-force delle celle dei
notebook.

FILE 01: Si mostra un'intro a google colab e come si può 
         utilizzare per lavorare con risorse HW potenti
         Lezione intro abbastanza inutile
FILE 02: Utilizzo di transformers tramite pipeline di
         HuggingFace, svolti vari esempi di varie task
         semplici giusto per far capire come funziona 
         l'utilizzo locale di questi modelli
FILE 03: Utilizzo di modelli in locale senza usare le
         pipeline pronte della libreria, facendo invece
         un deep dive più approfondito e spacchettando
         l'utilizzo nelle varie parti di una pipeline
